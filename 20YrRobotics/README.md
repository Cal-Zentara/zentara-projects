## What Robotic Parts Means
- Precision machined components (metal, tight tolerances)
- Gears, brackets, housings, shafts
- Made via CNC machining
- Materials: aluminum (default), steel (heavy/strong), plastics (light/cheap)

## Key Concepts Learned

**CNC = Computer Numerical Control**
A machine that cuts/shapes material controlled by a computer. The computer uses coordinates to tell the cutting tool exactly where to move.

**CNC Milling** — spinning tool carves across flat material
**CNC Turning** — material spins, tool stays still. Makes round parts.
**3-Axis** — cuts in 3 directions (simple parts)
**4-Axis** — adds rotation (medium complexity)
**5-Axis** — cuts from any angle (complex parts, higher cost)

**Materials:**
- Aluminum = lightweight, easy to cut, cheap (the default)
- Steel = stronger, heavier, more expensive to cut
- Plastics = lightest, cheapest, non-structural
